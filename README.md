# Thunmov

Free online movie website with free ads.

## Official Link

https://thunmov.vercel.app

## Technology

- Next 13
- TailwindCSS
- Swiper

## Features

- Fully responsive
- Save/delete favourite movies
- No popup ads
- Comments
- Socials sharing
- Support vietsub,...
